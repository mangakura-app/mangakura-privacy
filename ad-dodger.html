プライバシーポリシー

最終更新日：2026年6月17日

Ad Dodger（以下「本アプリ」）は、ユーザーの皆様のプライバシーを尊重します。 本ポリシーは、本アプリが収集・利用する情報について説明するものです。

1. 収集する情報

本アプリ自体はユーザーの個人情報を収集・保存・外部送信しません。端末内にはハイスコアのみ保存されます（外部サーバーへの送信なし）。

2. 広告について

本アプリは Google AdMob（Google LLC）による広告を表示します。広告の配信にあたり、AdMob はデバイス識別子や広告 ID、アプリの使用状況などの情報を収集することがあります。 収集される情報および利用目的については、Google のプライバシーポリシーをご確認ください。

  • Google プライバシーポリシー
  • AdMob のデータ収集に関する説明

3. 広告トラッキング（iOS）

iOS 14.5 以降では、App Tracking Transparency（ATT）フレームワークに基づき、パーソナライズ広告の配信に使用するトラッキング許可をお求めする場合があります。 許可しない場合でも本アプリは引き続きご利用いただけますが、表示される広告がパーソナライズされない場合があります。

4. サードパーティ SDK

本アプリは以下の SDK を使用しています。各 SDK のプライバシーポリシーも合わせてご確認ください。

  • Google Mobile Ads SDK（AdMob） — 広告配信
  • Google User Messaging Platform（UMP SDK） — 同意管理（GDPR 等）

5. 広告のオプトアウト

パーソナライズ広告を希望されない場合は、端末の設定から広告 ID のリセットまたは広告トラッキングの制限が可能です。

  • iOS：設定 → プライバシーとセキュリティ → トラッキング
  • Android：設定 → Google → 広告

6. プライバシーポリシーの変更

本ポリシーは必要に応じて改定することがあります。変更後のポリシーはこのページに掲載します。
